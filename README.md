# Placify---Job-Portal


Placify is a simple job and campus drive portal created to solve a common issue in colleges. Most placement updates are shared in WhatsApp groups, where important information gets lost between random messages. Because of this, students sometimes miss deadlines or key details.

Placify provides a clean and structured platform where students can find all placement-related information in one place without any confusion.

🚀 Features

📢 Campus Drive Updates – Company name, role, eligibility, and deadlines
🔍 Search & Filter Jobs
👤 Student Registration & Login
📝 Apply for Jobs directly through the portal
📊 Student Dashboard – View applied jobs
🔔 Basic Notifications for new drives
🧑‍💼 Admin Panel – Manage job postings

🛠️ Tech Stack

Frontend:HTML,CSS,JavaScript

Backend:Node.js,Express.js,Java (for backend logic / alternative implementation)
Database:MongoDB,local storage.

⚙️ Backend Details

The backend is mainly handled using Node.js and Express.js, which manages APIs, authentication, and database operations.Additionally, Java can be used for backend development (for example using Servlets or Spring Boot) to handle business logic, API creation, and scalability in larger systems.

Main Functionalities:
User Authentication (Login/Register)
Job Management (Add, Delete, View Jobs)
Application Tracking
Admin Controls
